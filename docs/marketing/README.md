# Marketing drafts

External-facing copy used to seed reach. Drafts only — review before
posting.

## Files

- [show-hn-draft.md](show-hn-draft.md) — Show HN title and first comment.
- [reddit-drafts.md](reddit-drafts.md) — Posts for r/ClaudeAI,
  r/LocalLLaMA, r/programming.
- [awesome-list-prs.md](awesome-list-prs.md) — Entry text and PR template
  for awesome-* lists.
- [devto-article-draft.md](devto-article-draft.md) — dev.to long-form
  article with title options and tags.
- [product-hunt-copy.md](product-hunt-copy.md) — Product Hunt launch copy
  and asset checklist.

## Suggested order

1. Land docs site at https://m24927605.github.io/ait/ first — every
   external link should point at a working page with full SEO meta.
2. Submit to 1–2 awesome-lists. Low effort, durable backlinks.
3. Post Show HN on a Tuesday or Wednesday morning Pacific time. Reply
   immediately with the prepared first comment.
4. Cross-post the Reddit drafts the same week, spaced one day apart per
   subreddit.
5. Publish dev.to article 3–5 days after Show HN, linking back to it.
6. Product Hunt launch last, once there is some Show HN / Reddit traffic
   to point at.

## What not to do

- Do not post the same body to multiple subreddits same day — Reddit
  spam-flags it.
- Do not edit the GitHub `description` or PyPI metadata mid-launch — link
  preview caches lock in whatever was live when the link was first shared.
- Do not promise features that are not yet shipped. Alpha is fine to
  state explicitly.
